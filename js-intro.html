JavaScript (JS) es un lenguaje de programación interpretado.

Se utiliza principalmente en su forma del lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas, aunque existe una forma de JavaScript del lado del servidor, como NodeJs. Su uso en aplicaciones externas a la web, por ejemplo en aplicaciones de escritorio y moviles es también significativo.

Desde el 2012, todos los navegadores modernos soportan completamente la version ECMAScript 5.1, la sexta edición (ECMAScript 6) se liberó en julio del 2015.



¿Donde colocarlo?

En HTML, el código JavaScript debe insertarse entre las etiquetas <script> y </script>


Javascript en <head> o <body>

Puede colocar cualquier cantidad de secuencias de comandos en un documento HTML.
Los scripts se pueden colocar en la sección <body>, o en la < head> de una página HTML, o en ambos.
También puede ser codigo embebido dentro de las etiquetas <script> y </ script> (como en el ejemplo anterior) o referenciar un codigo externo.


JavaScript en <head>


		  		<!DOCTYPE html>
		  		<html>
		  		<head>
		  		<!--  Javascript embebido -->
		  		<script>
			  		var x, y;
		  			x = 10; y = 5;
		  			prueba = x + y;
		  		</script>

		  		<!-- Javascript externo -->
		  		<script type="text/javascript" src="ejemplo.js"></script>
		  		</head>

		  		<body>

		  			<div>
		  				Contenido
		  			</div>


		  		</body>
		  		</html>

JavaScript en <body>


		  		<!DOCTYPE html>
		  		<html>
		  		<head>
		  		
		  		</head>

		  		<body>
		  			<div>
		  				Contenido
		  			</div>
		  			
		  		<!--  Javascript embebido -->
		  		<script>
			  		var x, y;
		  			x = 10; y = 5;
		  			prueba = x + y;
		  		</script>

		  		<!-- Javascript externo -->
		  		<script type="text/javascript" src="ejemplo.js"></script>
		  		</body>
		  		</html>

Sintaxis

La sintaxis se refiere a el conjunto de reglas que se siguen para escribir codigo en JavaScript.

		  		var x, y; // Declarar Variables
		  		x = 10; y = 5; // Asignando valores
		  		prueba = x + y; // Hacer una operación

Valores

La sintaxis de JavaScript define dos tipos de valores: literales y variables.

Literales

Los números se escriben con o sin decimales: 

		  		10.50

		  		10

Las cadenas de texto se escriben entre comillas simples o dobles: 

		  		"Maria"

		  		'Maria'

Para concatenar o unir cadenas de texto se utiliza el signo + como se ve a continuación: 

		  		// Unimos y agregamos un espacio de por medio
		  		"Maria" + " " + "Rodriguez"
		  		// También es posible agregar un espacio dentro de la misma cadena y luego unirlas
		  		"Maria " + "Rodriguez"

Identificadores

Los identificadores son nombres que se utilizan para identificar variables, palabras clave, funciones y etiquetas.

En JavaScript, el primer carácter debe ser una letra, un guión bajo (_) o un signo de dólar ($). Los caracteres subsiguientes pueden ser letras, dígitos, guiones bajos o signos de dólar.

Los números no están permitidos como el primer carácter, de esta forma, JavaScript puede distinguir fácilmente los identificadores de los números.

Todos los identificadores de JavaScript distinguen mayúsculas de minúsculas. JavaScript no reconoce VAR o Var, solo reconoce la palabra clave var

Las variables Nombre y nombre, son dos variables distintas: 

		  		var Nombre, nombre;
		  		Nombre = 'Jose';
		  		nombre = 'Miguel';

Camel Case

Los programadores utilizan diferentes formas de unir múltiples palabras en un nombre de variable: Guiones (-): Los guiones no estan permitidos en JavaScript. Se utilizan para realizar restas.

Guion bajo (_): primer_nombre, segundo_nombre.

Camel superior: PrimerNombre, SegundoNombre.

Camel Inferior: es el standar en el caso de los programadores de JavaScript, comienza con una letra minúscula y las demás palabras comienzan por mayúscula. Ejemplo: primerNombre, segundoNombre.



Comentarios

Los comentarios se utilizan generalmente para explicar el código escrito y hacerlo más entendible.

Los comentarios también se utilizan para evitar la ejecución de algún fragmento de código. Por ejemplo cuando probamos algún código, podemos comentar las partes que no queremos que se ejecuten.



Comentarios de línea

Los comentarios de línea comienzan con //. Cualquier texto que este despues de // hasta el final de la línea sera ignorado, no se ejecutará. 

		  		// Reciclamos el codigo anterior

		  		// Definimos las variables
		  		var Nombre, nombre;
		  		// Le Asignamos los Valores
		  		Nombre = 'Jose'; // valor Jose
		  		nombre = 'Miguel'; // valor Miguel

Comentarios de varias líneas

Los comentarios de línea comienzan con /* y terminan con */. Cualquier texto que este despues de /* y */ sera ignorado. 

		  		/* 
		  		Reciclamos el codigo anterior

		  		Definimos las variables 
		  		*/
		  		var Nombre, nombre;
		  		/* Le Asignamos los Valores */
		  		Nombre = 'Jose';
		  		nombre = 'Miguel';

JavaScript puede mostrar datos de diferentes maneras:

  • Escribir en un elemento HTML, usando innerHTML.
  • Escribir en el HTML, usando document.write().
  • Escribir en un cuadro de alerta, usando window.alert().
  • Escribir en la consola del navegador, usando console.log().


    • Usando innerHTML

    Para acceder a un elemento HTML, JavaScript puede usar el método document.getElementById(id).
    El atributo id define el elemento HTML. La propiedad innerHTML escribe el contenido dentro el elemento HTML seleccionado: 


    		  		<!DOCTYPE html>
		  		<html>
		  		<body>

		  		<h1>My First Web Page</h1>
		  		<p>My First Paragraph</p>

		  		<p id="demo"></p>

		  		<script>
		  		document.getElementById("demo").innerHTML = 5 + 6;
		  		</script>

		  		</body>
		  		</html>

    Usando document.write()

    Para propósitos de prueba, es conveniente usar document.write (): 


    		  		<!DOCTYPE html>
		  		<html>
		  		<body>

		  		<h1>Suma</h1>
		  		<p>5 + 6 =</p>

		  		<script>
		  		document.write(5 + 6);
		  		</script>

		  		</body>
		  		</html>

    Al usar document.write () después de que un documento HTML esté completamente cargado, se eliminará todo el HTML existente : 


    		  		<!DOCTYPE html>
		  		<html>
		  		<body>

		  		<h1>Suma</h1>
		  		<p>5 + 6 =</p>

		  		<button type="button" onclick="document.write(5 + 6)">Sumar</button>

		  		</body>
		  		</html>

    Usando window.alert()

    Puede usar un cuadro de alerta para mostrar datos: 


    		  		<!DOCTYPE html>
		  		<html>
		  		<body>

		  		
		  		<script>
		  		window.alert(5 + 6);
		  		</script>

		  		</body>
		  		</html>

    Usando console.log()

    Para fines de depuración, puede usar el método console.log () para mostrar datos.
    Estos datos apareceran en la consola de depuración del navegador. Puedes acceder a ella en la opción Herramienta para desarrolladores. 


    		  		<!DOCTYPE html>
		  		<html>
		  		<body>

		  		
		  		<script>
		  		console.log(5 + 6);
		  		</script>

		  		</body>
		  		</html>
