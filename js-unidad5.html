Cuando se escribe un código, a menudo se desea realizar diferentes acciones dependiendo de las distintas decisiones. Para eso se utilizan los condicionales.

En JavaScript tenemos las siguientes declaraciones condicionales:
  • if: para especificar si un código sera ejecutado, si la condición se cumple.
  • else: para especificar que un bloque de codigo se ejecutara, si la condicion en if no se cumple.
  • else if: si la primera condición falla, puede probar con otra condición.
  • switch: define muchas condiciones alternativas a ser ejecutados dependiendo de cual condición se cumpla.

Condicional if

Ejemplo
Considerando que x = 10.Imprima la variable x en una alerta si es mayor que 7: 


		  		if (x > 7){
		  		// se ejecuta el siguiento codigo ya que el resultado es verdadero
		  			alert(x);
		  		}

Condicional else

Ejemplo
Tomando el ejemplo anterior, pero ahora considerando que x = 5, imprima la variable x en una alerta si es mayor que 7 y sino muestre un mensaje: 


		  		if (x > 7){ 
		  		//este codigo no se ejecuta ya que el resultado es falso
		  			alert(x);
		  		} else {
		  		// se ejecuta el siguiente codigo
		  			alert("La variable x es menor que 7");
		  		}

Condicional else if

Ejemplo
Tomando el ejemplo anterior, pero ahora considerando que x = 7,imprima la variable x en una alerta si es mayor que 7 y si es igual a 7 envie un mensaje: 


		  		if (x > 7){ 
		  		//este codigo no se ejecuta ya que el resultado es falso
		  			alert(x);
		  		} else if (x == 7) {
		  		// se ejecuta el siguiente codigo
		  			alert("La variable x es igual a 7");
		  		}

Condicional switch

La instrucion switch se usa para realizar diferentes acciones en diferentes condiciones.

Asi es como se usa:

  • Pasamos una expresión o variable a switch().
  • El valor de la variable o expresión se compara con los valores de cada caso.
  • Si hay coincidencia, se ejecuta el bloque de codigo asociado.

Ejemplo
El método getDay() devuelve el día de la semana como un número entre 0 y 6 (Domingo = 0, lunes = 1, martes = 2, ...).

Este ejemplo usa el número del día de la semana para calcular el nombre del día de la semana: 


		  		var day = new Date().getDay();
		  		/*
		  			Tambien es posible realizarlo de la siguiente manera:

		  			switch(new Date().getDay())

		  			Es decir pasarle la expresión directamente..
		  		*/
		  		switch (day) {
		  		    case 0:
		  		        dia = "Domingo";
		  		        break;
		  		    case 1:
		  		        dia = "Lunes";
		  		        break;
		  		    case 2:
		  		        dia = "Martes";
		  		        break;
		  		    case 3:
		  		        dia = "Miércoles";
		  		        break;
		  		    case 4:
		  		        dia = "Jueves";
		  		        break;
		  		    case 5:
		  		        dia = "Viernes";
		  		        break;
		  		    case 6:
		  		        dia = "Sábado";
		  		}

Los ciclos pueden ejecutar un bloque de código varias veces.

Se utilizan a menudo cuando se trabajan con matrices.


Tipos


JavaScript admite diferentes tipos de bucles:

  • for: recorre un bloque de codigo varias veces.
  • while: ejecuta un bloque de codigo mientras que una condición especifica es verdadera, cuando es falsa se detiene.
  • do / while: también recorre un bloque de código mientras una condición especificada es verdadera


Ciclo for

Ejemplo 


			 	for (i = 0; i < 5; i++) {

			 	    text += "El numero es " + i + "
";

			 	}

  • La primera parte (i = 0), se ejecuta antes de que comience el ciclo.
  • La segunda parte (i < 5 ), define la condición para terminar el bucle.
  • La tercera parte(i++), se ejecuta cada vez que se ejecuta el ciclo.


    • Ciclo while

    Ejemplo
    En el siguiente ejemplo, el código se ejecutará una y otra vez, siempre que la variable i sea menor que 10: 


    			 	var i = 0;

			 	while (i < 10) {

			 	    text += "El numero es " + i;
			 	    i++;
			 	}

    Ciclo do / while

    El ciclo do / while, a diferencia de el ciclo while, este ejecutará el bloque de código una vez, antes de verificar si la condición es verdadera, luego actua como el ciclo while ejecutando el ciclo siempre que la condición sea verdadera.
    Esto quiere decir que el ciclo do / while, siempre se ejecutara al menos una vez.

    Ejemplo
    El siguiente ejemplo usa un ciclo do / while. El ciclo siempre se ejecutará al menos una vez, incluso si la condición es falsa, porque el bloque de código se ejecuta antes de que se pruebe la condición: 


    			 	var i = 10;

			 	do {
			 	    text += "El numero es " + i;
			 	    i++;
			 	}
			 	while (i < 10);

    NOTA!

    ¡En los ciclos while y do / while, no olvides aumentar la variable utilizada en la condición, de lo contrario, el ciclo nunca terminará!
