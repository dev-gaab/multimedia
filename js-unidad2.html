Variables y Tipos de datos


Las variables en JavaScript pueden contener muchos tipos de datos: números (number), cadenas de texto (string), booleanos (boolean), arreglos (array) y objetos (object).
Estos no son todos los tipos de datos, pero si los mas utilizados. Por ejemplo existe el tipo de dato indefinido (undefined), nulo (null) y función (function). 

			var numero = 20; // Variable tipo number(numerico)
			var cadena = "José"; // Variable tipo string(cadena de texto)
			var verdadero = true; // Variable tipo boolean( Booleano igual a verdadero)
			var falso = false; // Variable tipo boolean( Booleano igual a falso)
			var arreglo = ["José", "María", "Jesus"]; // Variable tipo array (Arreglo de cadenas de caracteres)
			var objeto = {nombre: "José", apellido: "Hernandez", edad: 30}; // Variable tipo object (Objeto)
			var carro; var casa = undefined; // Variables tipo undefined (indefinidas)
			var persona =  null; // Variable tipo null (nula)
			var funcion = function miFunc(); // Variable tipo function (funcion)

En la programación, los tipos de datos son un concepto importante. Sin tipos de datos , una computadora no puede resolver esto: 


			var x = 20 + "Luis";

¿Tiene algun sentido sumar 20 a Luis?. JavaScript tratara el ejemplo anterior como una cadena: 


			var x = 20 + "Luis"; // Resultado = 20Luis
			var y = "20" + "Luis"; // Resultado = 20Luis

Esto sucede debido a que JavaScript utiliza el signo + tanto para sumar como para concatenar variables.
Además JavaScript toma las operaciones de izquierda a derecha. Veamos el ejemplo: 


			var x = 20 + 5 + "Luis"; // Resultado = 25Luis
			var y = "Luis" + 20 + 5; // Resultado = Luis205

En el primer ejemplo, JavaScript trata 20 y 5 como números, hasta que llega a "Luis".
En el segundo ejemplo, dado que el primer operando es una cadena, todos los operandos se tratan como cadenas.


Los tipos de datos son dinámicos


JavaScript tiene tipos de datos dinámicos. Esto quiere decir que la misma variable puede cambiar de tipo, dependiendo del dato que contenga en ese momento. 

			var x; // Es undefined
			x = null; // Ahora es null
			x = 5; // Ahora es number
			x = "Luis"; // Ahora es string

Operador typeof


Puede usar el operador typeof para encontrar el tipo de una variable JavaScript.
El operador typeof devuelve el tipo de una variable o una expresión: 

			typeof "Luis" // Retorna string
			typeof 50 // Retorna number
			typeof ["Jose", "Hernandez"] // Retorna array
			typeof {nombre: "Jose", apellido: "Hernandez"} // Retorna object
