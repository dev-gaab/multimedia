Cuando se escribe un código, a menudo se desea realizar diferentes acciones dependiendo de las distintas decisiones. Para eso se utilizan los condicionales.

En JavaScript tenemos las siguientes declaraciones condicionales:
  • if: para especificar si un código sera ejecutado, si la condición se cumple.
  • else: para especificar que un bloque de codigo se ejecutara, si la condicion en if no se cumple.
  • else if: si la primera condición falla, puede probar con otra condición.
  • switch: define muchas condiciones alternativas a ser ejecutados dependiendo de cual condición se cumpla.

Condicional if

Ejemplo
Considerando que x = 10.Imprima la variable x en una alerta si es mayor que 7: 


      if (x > 7){
      // se ejecuta el siguiento codigo ya que el resultado es verdadero
        alert(x);
      }

Condicional else

Ejemplo
Tomando el ejemplo anterior, pero ahora considerando que x = 5, imprima la variable x en una alerta si es mayor que 7 y sino muestre un mensaje: 


      if (x > 7){
      //este codigo no se ejecuta ya que el resultado es falso
        alert(x);
      } else {
      // se ejecuta el siguiente codigo
        alert("La variable x es menor que 7");
      }

Condicional else if

Ejemplo
Tomando el ejemplo anterior, pero ahora considerando que x = 7,imprima la variable x en una alerta si es mayor que 7 y si es igual a 7 envie un mensaje: 


      if (x > 7){
      //este codigo no se ejecuta ya que el resultado es falso
        alert(x);
      } else if (x == 7) {
      // se ejecuta el siguiente codigo
        alert("La variable x es igual a 7");
      }

Condicional switch

La instrucion switch se usa para realizar diferentes acciones en diferentes condiciones.

Asi es como se usa:

  • Pasamos una expresión o variable a switch().
  • El valor de la variable o expresión se compara con los valores de cada caso.
  • Si hay coincidencia, se ejecuta el bloque de codigo asociado.

Ejemplo
El método getDay() devuelve el día de la semana como un número entre 0 y 6 (Domingo = 0, lunes = 1, martes = 2, ...).

Este ejemplo usa el número del día de la semana para calcular el nombre del día de la semana: 


      var day = new Date().getDay();
      /*
        Tambien es posible realizarlo de la siguiente manera:

        switch(new Date().getDay())

        Es decir pasarle la expresión directamente..
      */
      switch (day) {
          case 0:
              dia = "Domingo";
              break;
          case 1:
              dia = "Lunes";
              break;
          case 2:
              dia = "Martes";
              break;
          case 3:
              dia = "Miércoles";
              break;
          case 4:
              dia = "Jueves";
              break;
          case 5:
              dia = "Viernes";
              break;
          case 6:
              dia = "Sábado";
      }
