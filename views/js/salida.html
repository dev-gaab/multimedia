JavaScript puede mostrar datos de diferentes maneras:

  • Escribir en un elemento HTML, usando innerHTML.
  • Escribir en el HTML, usando document.write().
  • Escribir en un cuadro de alerta, usando window.alert().
  • Escribir en la consola del navegador, usando console.log().


    • Usando innerHTML

    Para acceder a un elemento HTML, JavaScript puede usar el método document.getElementById(id).
    El atributo id define el elemento HTML. La propiedad innerHTML escribe el contenido dentro el elemento HTML seleccionado: 


                 <!DOCTYPE html>
             <html>
             <body>

             <h1>My First Web Page</h1>
             <p>My First Paragraph</p>

             <p id="demo"></p>

             <script>
             document.getElementById("demo").innerHTML = 5 + 6;
             </script>

             </body>
             </html>

    Usando document.write()

    Para propósitos de prueba, es conveniente usar document.write (): 


                 <!DOCTYPE html>
             <html>
             <body>

             <h1>Suma</h1>
             <p>5 + 6 =</p>

             <script>
             document.write(5 + 6);
             </script>

             </body>
             </html>

    Al usar document.write () después de que un documento HTML esté completamente cargado, se eliminará todo el HTML existente : 


                 <!DOCTYPE html>
             <html>
             <body>

             <h1>Suma</h1>
             <p>5 + 6 =</p>

             <button type="button" onclick="document.write(5 + 6)">Sumar</button>

             </body>
             </html>

    Usando window.alert()

    Puede usar un cuadro de alerta para mostrar datos: 


                 <!DOCTYPE html>
             <html>
             <body>


             <script>
             window.alert(5 + 6);
             </script>

             </body>
             </html>

    Usando console.log()

    Para fines de depuración, puede usar el método console.log () para mostrar datos.
    Estos datos apareceran en la consola de depuración del navegador. Puedes acceder a ella en la opción Herramienta para desarrolladores. 


                 <!DOCTYPE html>
             <html>
             <body>


             <script>
             console.log(5 + 6);
             </script>

             </body>
             </html>
