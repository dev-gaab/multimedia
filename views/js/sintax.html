Sintaxis

La sintaxis se refiere a el conjunto de reglas que se siguen para escribir codigo en JavaScript.

             var x, y; // Declarar Variables
             x = 10; y = 5; // Asignando valores
             prueba = x + y; // Hacer una operación

Valores

La sintaxis de JavaScript define dos tipos de valores: literales y variables.

Literales

Los números se escriben con o sin decimales: 

             10.50

             10

Las cadenas de texto se escriben entre comillas simples o dobles: 

             "Maria"

             'Maria'

Para concatenar o unir cadenas de texto se utiliza el signo + como se ve a continuación: 

             // Unimos y agregamos un espacio de por medio
             "Maria" + " " + "Rodriguez"
             // También es posible agregar un espacio dentro de la misma cadena y luego unirlas
             "Maria " + "Rodriguez"

Identificadores

Los identificadores son nombres que se utilizan para identificar variables, palabras clave, funciones y etiquetas.

En JavaScript, el primer carácter debe ser una letra, un guión bajo (_) o un signo de dólar ($). Los caracteres subsiguientes pueden ser letras, dígitos, guiones bajos o signos de dólar.

Los números no están permitidos como el primer carácter, de esta forma, JavaScript puede distinguir fácilmente los identificadores de los números.

Todos los identificadores de JavaScript distinguen mayúsculas de minúsculas. JavaScript no reconoce VAR o Var, solo reconoce la palabra clave var

Las variables Nombre y nombre, son dos variables distintas: 

             var Nombre, nombre;
             Nombre = 'Jose';
             nombre = 'Miguel';

Camel Case

Los programadores utilizan diferentes formas de unir múltiples palabras en un nombre de variable: Guiones (-): Los guiones no estan permitidos en JavaScript. Se utilizan para realizar restas.

Guion bajo (_): primer_nombre, segundo_nombre.

Camel superior: PrimerNombre, SegundoNombre.

Camel Inferior: es el standar en el caso de los programadores de JavaScript, comienza con una letra minúscula y las demás palabras comienzan por mayúscula. Ejemplo: primerNombre, segundoNombre.



Comentarios

Los comentarios se utilizan generalmente para explicar el código escrito y hacerlo más entendible.

Los comentarios también se utilizan para evitar la ejecución de algún fragmento de código. Por ejemplo cuando probamos algún código, podemos comentar las partes que no queremos que se ejecuten.



Comentarios de línea

Los comentarios de línea comienzan con //. Cualquier texto que este despues de // hasta el final de la línea sera ignorado, no se ejecutará. 

             // Reciclamos el codigo anterior

             // Definimos las variables
             var Nombre, nombre;
             // Le Asignamos los Valores
             Nombre = 'Jose'; // valor Jose
             nombre = 'Miguel'; // valor Miguel

Comentarios de varias líneas

Los comentarios de línea comienzan con /* y terminan con */. Cualquier texto que este despues de /* y */ sera ignorado. 

             /*
             Reciclamos el codigo anterior

             Definimos las variables
             */
             var Nombre, nombre;
             /* Le Asignamos los Valores */
             Nombre = 'Jose';
             nombre = 'Miguel';
