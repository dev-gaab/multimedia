Funciones


Una función JavaScript es un bloque de código diseñado para realizar una tarea en particular. Una función se ejecuta cuando "algo "la invoca (la llama).


Sintaxis


Una función de JavaScript se define con la palabra clave function, seguida de un nombre, seguido de paréntesis ().
Los nombres de funciones pueden contener letras, números, guiones bajos y signos de dólar (las mismas reglas que las variables).
Los paréntisis pueden incluir nombres de parámetros separados por comas: (parámetro1, parámetro2, ...).
El codigo que se ejecutará, por la función, se coloca entre llaves: {} 


      function nombre(parametro1, parametro2)
      {
        // Codigo a ejecutar
      }

Función de retorno


Cuando JavaScript alcanza una declaración de retorno (return), la función dejara de ejecutarse.
Las funciones a menudo calculan un valor de retorno. El valor de retorno es "devuelto" a quien "la llama".
Ejemplo:
Calcule el producto de dos números y devuelva el resultado: 


      var a = multiplicar(5, 5);

      function multiplicar(x, y){
        return x * y;
      }

El resutado almacenado en a será: 25.


¿Por qué funciones?


Puede reutilizar el código, se define la función una vez y se puede usar las veces que se requiera en cualquier parte del codigo.

El operador () invoca la función. Llamar a la funcion sin () devolverá la definición de la función en lugar del resultado, es decir, devolvera el código.

Reutilizando el código anterior llamaremos a la función multiplicar sin los (): 


      var a = multiplicar;

      function multiplicar(x, y){
        return x * y;
      }

El resultado almacenado en a será el siguiente: 

      function multiplicar(x, y){
        return x * y;
      }
